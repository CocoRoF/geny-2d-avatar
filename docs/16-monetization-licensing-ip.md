# 16. 수익화, 라이선스, IP (Monetization, Licensing & IP)

> **한 줄 요약**: "AI 생성 툴" 의 성패는 기술보다 **법과 신뢰**에서 갈린다. 누가 뭘 소유하고, 누가 뭘 쓸 수 있고, 분쟁이 나면 어떻게 되는지 — 이 문서가 그 기준이다. **결정을 미루지 않는다**.

> **고지**: 본 문서는 제품·비즈니스 설계 기준을 담으며, 법률 자문을 대체하지 않는다. 최종 약관은 각 지역 법률 자문을 거친다.

---

## 1. 수익화 개요 (Monetization Overview)

### 1.1 가치 교환의 구조

| 누가 가치를 주는가 | 무엇을 받는가 |
|---|---|
| 사용자 | 내 캐릭터를 빠르게, 싸게, 움직이게 만드는 능력 |
| 플랫폼 | 도구, 파이프라인, AI 오케스트레이션, 커뮤니티, 배포 |
| 템플릿/프리셋 제작자 | 자신의 디자인 작업을 재사용 가능한 자산으로 판매 |
| 엔터프라이즈 | 안정성, 통제, SLA, 통합 |
| IP 권리자 | 자신의 스타일/IP 를 기반으로 한 배포 권리 |

### 1.2 핵심 BM 축

1. **Freemium 구독**: 개인·팀 월 플랜.
2. **사용량 기반 크레딧**: AI 호출·export 기반.
3. **마켓플레이스 수수료**: 템플릿/프리셋/모션팩 거래.
4. **엔터프라이즈 라이선스**: SSO, 보안, SLA, 전용 쿼터.
5. **임베드/SDK 라이선스**: B2B 채팅·커머스 통합.

## 2. 플랜 구조 (Plans)

### 2.1 플랜 개요 (초안)

| 플랜 | 대상 | 월요금 | 크레딧 | 특징 |
|---|---|---|---|---|
| **Free** | 체험 | 0 | 30 crd | 저비용 모델, 해상도 1024, 비상업 |
| **Starter** | 개인 크리에이터 | $15 | 300 crd | 2048, 상업 허용, VTS/Web |
| **Pro** | 파워 유저 | $45 | 1,200 crd | 4096, PRO 품질, 배치 10대/월 |
| **Team** | 소규모 스튜디오 (5석) | $180 | 5,000 crd | 공유/리뷰, 브랜딩 |
| **Enterprise** | 법인 | 협의 | 협의 | SSO, 감사, SLA, 전용 쿼터 |

> 수치는 초안. β 의 실제 비용 데이터로 재조정.

### 2.2 크레딧 환산 원칙

- 1 크레딧 ≈ 파츠 AI 호출 1회에 해당. 아바타 1대 생성 ≈ 25 크레딧(표준 해상도).
- Export 1회 ≈ 2 크레딧.
- 배치/야간 가격은 할인.

### 2.3 추가 구매 (Top-up)

- 크레딧 팩: 200/1,000/5,000.
- 롤오버: 60일. 사용자 혼란 방지로 기한 공지 강화.

## 3. 라이선스 프레임워크 (Licensing Framework)

### 3.1 4 종류 권리

1. **Input License** — 업로드 레퍼런스/스타일에 대한 권리(사용자 → 플랫폼 일부 허가).
2. **Output License** — 생성 결과에 대한 권리(사용자 소유, 플랫폼 일부 사용권).
3. **Template/Preset License** — 공식/커뮤니티 자산 사용 허가.
4. **Third-party AI License** — 외부 모델 벤더의 이용 약관.

### 3.2 기본 약속

- **사용자가 만든 아바타의 상업 이용권은 사용자에게**.
- **플랫폼은 서비스 제공 목적의 처리 권한** 만 보유.
- **학습 데이터 재사용**: 사용자가 **옵트인** 해야만 허용. 기본 off.
- **지우면 지워진다**: 30일 내 파기(법적 보존 의무 제외).

### 3.3 Input License 상세

- 사용자는 업로드 시 해당 자료에 대한 **적법한 이용 권리** 를 선언.
- 타인 캐릭터/사진/상표 업로드는 **사용자 책임** 을 강조.
- 플랫폼은 악용 탐지 및 삭제권 보유.

### 3.4 Output License 상세

- **사용자 소유**: 상업 이용 가능(플랜 제한 내).
- **책임 전가 금지**: 플랫폼 표기 강요 없음(플랜별 엔터프라이즈 옵션).
- 라이선스 증명서(`license.json`) 자동 동봉(11 §9).

### 3.5 Template/Preset License (마켓)

- **비상업 무료 / 상업 유료** 두 트랙.
- 2차 창작 허용 여부 태그: `remix-allowed`, `no-remix`.
- 만든이 크레딧 자동 표기(옵션).

### 3.6 Third-party AI License

- nano-banana 등 외부 모델의 이용 약관 준수.
- 모델 별 "상업 이용 가능 여부" 를 UI 에서 **사전 표기**.
- 약관 변경 시 **가격/가이드 업데이트** 48시간 내 공지.

## 4. IP 보호 (IP Protection)

### 4.1 "스타일" 의 취급

- 사용자가 업로드한 레퍼런스로 만든 스타일 프로파일은 **사용자 소유**.
- 다만, 그 스타일이 특정 작가의 고유 스타일과 유사할 경우 **공개/유통** 에 제한이 붙을 수 있다.
  - 예: 퍼블릭 마켓 공개 전 **유사도 검사** 필터.
  - 사용자는 자기 스타일이 침해 의심을 받으면 **소명** 가능.

### 4.2 스타일 등록제 (Style Registry)

- 공개 상업적 사용을 원하는 스타일은 **공식 등록** 가능.
- 등록: 작가 본인 선언 + 증빙(포트폴리오/저작권).
- 등록된 스타일은 **침해 신고**의 1차 대상.

### 4.3 DMCA / 침해 신고

- 신속 처리 채널: 웹 폼 + 이메일.
- 접수 후 24–72시간 내 1차 검토.
- 허위 신고 반복 시 제재.

### 4.4 워터마크 & 프로비넌스

- 내부적으로 모든 생성물에 **소프트 워터마크(C2PA 호환)** 삽입 옵션.
- 지워져도 **프로비넌스 서명** 이 남아 검증 가능.

## 5. 콘텐츠 정책 (Content Policy)

### 5.1 금지 범주

- 미성년 대상 성적 표현.
- 실존 인물의 비동의 생성(딥페이크).
- 폭력/증오 선동.
- 타인 상표/IP 명시적 도용.
- 불법·사기 목적.

### 5.2 회색 영역

- 만화적 폭력, 성인향 표현(플랜/연령 게이팅 하에서만).
- 공인 캐릭터 패러디(풍자 범위).

### 5.3 집행

- 자동 필터 + 사람 리뷰.
- 위반 시 삭제·계정 경고·영구 차단.
- 반복 위반자: 법적 대응 가능.

## 6. 엔터프라이즈 라이선스

### 6.1 표준 계약 항목

- 이용 범위(인원, 사이트).
- SLA (가용성, 응답 시간).
- 데이터 소유(고객 데이터는 고객 소유).
- 보안/감사: SOC2 준수 목표, 침해 통보 72h.
- 지원: 전용 채널, 우선 처리.
- 데이터 거버넌스: 지역, 삭제.

### 6.2 가격

- 기본 Flat + 사용량 상한. 사용량 초과는 할증.
- 커스텀: 자체 GPU 격리, 전용 템플릿, 화이트 라벨.

### 6.3 화이트 라벨/재판매

- Agency/스튜디오는 **재판매 계약** 가능.
- 고객사 최종 사용자에게 플랫폼 노출 여부 협상.

## 7. 커뮤니티 마켓플레이스

### 7.1 거버넌스

- 등록 심사(저작권 동의, 품질).
- 스타일 등록제와 연계.
- 민감 카테고리 제한(16 §5 정책).

### 7.2 수익 분배

- 초안: 판매자 70% / 플랫폼 30%.
- 지역 세금은 플랫폼이 징수/대납.
- 정산 주기: 월 1회.

### 7.3 분쟁

- 구매자 보호: 7일 환불(디지털 자산 특성 고려).
- 저작권 분쟁: 보류 + 조사.

### 7.4 품질 관리

- 자동 검수 통과 필수.
- 커뮤니티 리뷰(별점/후기).
- 다운로드 수/환불률 기반 자동 강등/부각.

---

## 8. 데이터 거버넌스 & 개인정보

### 8.1 원칙

- **최소 수집**.
- **명시적 목적 구속**.
- **이동 가능성**(사용자가 자기 데이터를 받아 떠날 수 있음).
- **잊힐 권리**.

### 8.2 규제 준수

- 한국 개인정보보호법, GDPR, CCPA.
- 아동 보호 법령.
- 데이터 전송: SCC/DPA 체결.

### 8.3 모델 학습 사용

- 사용자 데이터의 **모델 학습 사용은 옵트인**.
- 학습 포함 시 **크레딧 할인/혜택** 제공(인센티브).
- 익명화·집계 수준의 통계 사용은 약관 내 허용(서비스 개선 목적).

## 13. 분쟁 해결

- **중재 우선**: 관할 법원 전에 내부 중재 30일.
- **커뮤니티 분쟁**: 공공 후기/평가는 허용(허위/악성 제외).
- **환불 분쟁**: CS 1차 → 관리자 에스컬 → 결제사 협조.

## 14. 가격·정책 변경 관리

- 변경은 **30일 전 공지**(엔터프라이즈는 계약서 우선).

- 정책 변경 이력은 퍼블릭 페이지에 보관.

## 16. 크레딧·쿼터·API

### 16.1 API 가격

- API 호출은 같은 크레딧 경제.
- 엔터프라이즈: 전용 API 키/쿼터/SLA.

### 16.2 SDK 임베드

- 우리 Web SDK 는 무료 배포.
- 임베드로 발생하는 호출은 소유 계정 크레딧에서 차감.
- 광고/브랜딩 삽입 금지(엔터 화이트 라벨 제외).
